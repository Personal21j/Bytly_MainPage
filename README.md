# Bytly Website

Official website for **Bytly**, a restaurant discovery platform built around recommendations from people you know and trust.

**Live Website:**  
https://personal21j.github.io/Bytly_MainPage/

---

## About

Bytly reimagines restaurant discovery by focusing on recommendations from friends instead of anonymous reviews. The platform helps users discover new places, share genuine experiences, and organize restaurants they want to visit in a simple, map-first interface.

---

## Website

This repository contains the source code for the official Bytly website, which includes:

- Landing page
- Product overview
- Feature showcase
- Support page
- Privacy Policy

The website serves as the public-facing introduction to the Bytly platform and provides information about the product and its upcoming launch.

---

## Core Features

Bytly is designed around a social approach to restaurant discovery and includes features such as:

- Interactive map-based restaurant discovery
- Recommendations from friends
- Restaurant reviews with ratings, photos, and notes
- Personal wishlists and custom collections
- Saved favorite restaurants
- A clean and intuitive user experience
